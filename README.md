# Exact432HzPlayerWindows
Exact 432Hz Player for Windows

### Introduction
A windows platform 432hz player to analyzes the instrument frequencies from the audio file (PCM raw data) and automatically adjusts the playback pitch to convert all music to a 432Hz tone.

This APP is based on BASS audio Library: https://www.un4seen.com/
